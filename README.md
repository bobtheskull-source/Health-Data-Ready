# Health Data Ready

**MVP for Washington MHMDA Compliance Platform**

A lightweight compliance platform for small medical-adjacent businesses navigating Washington State's My Health My Data Act (MHMDA).

## About

Health Data Ready helps Washington businesses determine MHMDA applicability, inventory consumer health data, identify vendors/processors, generate required policy documents, and track compliance evidence.

**Not legal advice.** This platform produces operational outputs for review by qualified counsel.

## Current Status

This repository is under active development. The MVP is being built milestone by milestone:

- **M0**: Foundation (auth, RBAC, audit, security baseline)
- **M1**: Onboarding + Applicability assessment
- **M2**: Data inventory + Vendor register
- **M3**: Policy generator
- **M4**: Rights request workflow
- **M5**: Evidence vault + Export bundle
